# PC Builder

An interactive web application for building and customizing your own PC.

## Features

- Select CPU, GPU, RAM, SSD, and PSU components
- Real-time compatibility checking
- Total price calculation
- Gaming performance estimation

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Run

Simply open `index.html` in your browser.
